RoggenData RoggenSensor::getData (  )  [virtual]

Implements the getData() method from the IntSource interface class. The method blocks when there is no data available. Therefore from the outside it looks and feels the same whether you use a single sensor (RoggenSensor) of a fusion (RoggenSensorFusion) of them. Both classes implement the same interface IntSource. The method calls the FrameParser from Daniel Roggen to convert the sensor data.

Implements IntSource.
