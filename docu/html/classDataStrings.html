string DataStrings::encodeByte ( unsigned int  num  )  [static]

Helper method to encode a byte for sending as a text. In the output string only the characters from A to P are used. Therefore one byte takes two characters to store. In return we have loads of escape characters left and we don't have to be afraid of eof occuring in the data stream. Values bigger then 255 aren't accepted. In other words only the lowest 8 Bits are wanted. Unsigned numbers are expected!
